jQuery Finger

Try it

Download Github

jQuery Finger unifies click and touch events by removing the 300ms delay on touch devices. It also provide a common set of events to handle basic gestures such as drag and pinch. Small (< 1kb gzipped), it is focused on performance, is well tested and ... also supports jQuery delegated events.

Getting started

You can give it a shot by adding the script right after jQuery: 


<script src="https://raw.github.com/ngryman/jquery.finger/v0.0.9/dist/jquery.finger.js"></script>

Then you can subscribe to any of the jQuery Finger events like this: 


// direct event
$('.touch').on('tap', function(e) {
    console.log(this, e);
});

// delegated event
$('.parent').on('tap', '.touch', function(e) {
    console.log(this, e);
});

Events

jQuery Finger tries to focus on a small, simple set of events. You can use the provided events like any other jQuery event.

Tap

The tap event is the most basic one and is essentially like a click. On touch devices, it removes the 300ms delay of the standard click.

jQuery Finger won't fire this event if the pointer has moved above a specific threshold (see Configuration). 


// for direct event
$('.touch').on('tap', function(e) {
    console.log(this, e);
});

// for direct event
$('.parent').on('tap', '.touch', function(e) {
    console.log(this, e);
});

Configuration


$.Finger = {
    pressDuration: 300,
    doubleTapInterval: 300,
    flickDuration: 150,
    motionThreshold: 5
};
